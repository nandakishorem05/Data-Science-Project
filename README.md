# 📱 Cellphone Price Prediction

## 📌 Project Overview

This project predicts the **price range of a cellphone** using Machine Learning.
The model analyzes different mobile specifications such as battery power, RAM, screen size, and camera features to estimate the price category.

---

## 🧠 Machine Learning Model

The project uses the **K-Nearest Neighbors (KNN)** algorithm to classify cellphone prices.

---
